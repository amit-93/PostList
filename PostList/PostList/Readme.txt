
Project: This project shows the list of Post data in a grid. It show only Post ID and Post Title and when user select any single row it create a new section which contains the detail of selected row.
         User can copy the detail from new section in three format : Plain, Json or Html. 
 
Technologies used: C#, WPF, Rest API
 
Dependencies: One dependency on external dll "Newtonsoft.Json" to de-serialize the returned data from Rest API into desired format.
